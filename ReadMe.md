## Features

Các chức năng chính của ứng dụng:
1. Đọc sách file PDF.
2. Đổi Theme: Dark - Light Mode.
3. Thêm, sửa, xóa, truy cập BookMark.
4. Tải File từ Internet, Upload File từ PC.


## Requirements

- Java 11
- Maven
- PDBox
- FlatLaf
